문제의 키워드는 캐시인데 시간을 두고 다시 코딩해봐야겠습니다. 
길이는 계산하는 재귀함수를 작성하다가 다음 수를 계산하는 메소드와 
길이(수를 세는) 메소드를 분리하였습니다. 재귀를 썼다고 착각했습니다. 
그리고 전역변수를 사용하려다가 변수인지 메소드인지 
구분하지 못한다는 오류 메시지가 있어서 인스턴스 변수를 써야할까 생각했습니다. 
실행시간을 줄이기 위해 3*n + 1 을 n + n + n + 1 로 바꾸니 조금 더 빨라졌습니다. 
다른 분 소스를 보니 배열에 쭉 넣어놓고 max 메소드를 쓰면 하나씩 비교하지 
않더라도 편하게 쓸 수 있었습니다.

로라 리뉴얼 프로젝트에 루비 코드파일을 업로드하면 서버에서 실행하여 실행결과와 
시간을 측정하는 기능을 추가하면 좋겠습니다.
